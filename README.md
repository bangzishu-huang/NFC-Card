# NFC-Card
A contact card I can bring with me whenever, wherever, and connect with other people.

I wanted to make this as a little side project that would be pretty cool to have around. This also served as a learning experience for me to be educated in EasyEDA, because I have only been using KiCAD for schematics and PCB design.
